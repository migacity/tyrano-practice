## スクリプトの構成

* 画像の差し替えくらいでほとんどそのまま必要なファイル
    - **tyrano.ks**: 標準関数群
    - **first.ks**: エントリポイント
    - **title.ks**: タイトル画面
    - **make.ks**: セーブデータロード時に呼ばれる特殊なファイル
    - **config.ks**: 設定画面
* 場合によっては必要なファイル
    - **replay.ks**: 回想モード画面
    - **cg.ks**: CG一覧画面
* メインのストーリーファイル
    - **scene1.ks**: メインストーリー

## tyrano.ksに含まれる関数
